# Enterprise Knowledge Assistant

## Project Overview

Enterprise Knowledge Assistant is a production-grade Retrieval-Augmented Generation (RAG) application that allows users to ask questions about private documents using natural language.

The system retrieves relevant information from uploaded documents and generates accurate answers with source citations using a Large Language Model (LLM).

This project is being developed as a complete end-to-end AI Engineering project, covering everything from system design and implementation to deployment and monitoring.

---

## Project Goals

- Learn AI Engineering
- Build a Production-Grade RAG System
- Understand FastAPI
- Learn Vector Databases
- Learn Embeddings
- Deploy an AI Application
- Build a Portfolio Project
